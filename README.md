# Stimuludo

### [stimuludo.com](https://stimuludo.com) is a website designed to stimulate and train cognitive abilities in a fun and engaging way.

---

## Technical Stack

- **Front-End**: HTML5, CSS3, JavaScript (Vanilla)
- **Back-End**: Node.js, Express.js
- **Database**: MySQL, Sequelize ORM
- **Real-Time**: Socket.io
- **Security**: JWT, bcrypt

## Features

- **Authentication**: Secure registration/login (bcrypt, JWT).
- **Data**: Statistics tracking and match history in a relational database.
- **Multiplayer**: WebSockets implementation for real-time interactions.
